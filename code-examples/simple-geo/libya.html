Libya Attacks API

The Libya attacks data is available on SimpleGeo’s storage API.

SimpleGeo provide a few client libraries including a JavaScript library.

Alternatively you can hack around with the REST API itself by appending token=dPCyPfyxGDP85j7ztP5PGMcWCwY8AhFc to any of these endpoints. For example:

https://api.simplegeo.com/0.1/records/libya/nearby/32.895,13.28.json?token=dPCyPfyxGDP85j7ztP5PGMcWCwY8AhFc&radius=100&limit=100
